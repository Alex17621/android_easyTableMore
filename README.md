# android_easyTableMore 安卓开发的简单课程表
**目录（按功能模块划分）**
**(仿微信，登陆后左右滑动切换界面)**
- 登录界面
	- （账号密码必须相同，长度必须为9位）
	- 记住密码功能
- 课程主界面（默认此界面，可以修改）
	- 增加课程
	- 查看课程
		- 课程里有对应的笔记信息（添加笔记时标签属性对应的就是课程名字）
	- 删除课程
	- 修改课程
- 我的界面
	- 查看或修改学期
	- 删除所有课程
	- 退出账号
- 笔记界面
	- 查询笔记（模糊查询，查询关键字包括：标题、标签、文本内容）
	- 添加笔记
	- 查看笔记
	- 修改笔记
	- 长按删除笔记
